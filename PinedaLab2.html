• <tr>: It represents a tabe row and multiple is used in order to create table rows.
  • <th>: It defines a cell header within the tabe and is used to lable table rows and data in the table.
  • <td>: It defines data within the table and is used to show data inside the table of each row.
  • <thead>: It is used for group header content in a table. You need to have <tr> in order to make things more specific
  • <tfoot>: It is used for group footer content in a table. You need to have <tr> inside the <tfoot>: in order to specify each part of the table
  • <caption>:; It helps provide a caption for the table and is used right after the tabe tag in oder to help explain what the table is

Colspan vs Rowspan: Colspan is used to help expand a column in a cell by a certain number, while rowspan helps expand a row by a certain number

Cells spacing vs padding: Cells spacing helps define the space between cells in the table while padding defines the space between the border and the cells content.

Defining borders: You can use the "border" in order to define the table borders which targets <th> and <td>. To control the border-width, you can use "border-width:__px".
